This website has been created as a final team project in GoIT-PyWeb7 course. It consists of 6 applications.
    {# Auth block #}
  • Authentication
    • To use this website a user must be authenticated. The only page that can be accessed by an unauthorized user is this page and a sign-up page.

    On the sign-up page the user is able to sign up and sign in using modal login form on the top right corner. Also in that corner the user can sign out.

    {# Contacts block #}
  • Contacts
    • Every user can keep a book of their contacts in Contacts section. Each contact has a name, an address, a birthdate, gender, and a list of emails and phones. A new contact can be added using the button Add New Contact. Then the contact can be deleted or updated by clicking on the corresponding buttons next to it on the Contacts page. The user can also filter the entries and get a list of people who have birthdays in the next N days.
    {# File storage block #}
  • File storage
    • The user is able to upload files with valid extensions. The list of valid extensions can be found in the header on the files page. Just click on the button Extensions. Uploaded files are separated into categories according to their extensions and stored in the DropBox cloud. The user also can filter files by category.
    {# Notes block #}
  • Notes
    • The user is able to add notes by clicking on the button Add note. Each note has to include a tag that describes the topic and the subject of the entry. In the Tags section, you can create new tags and delete those that are not used. In the list of notes, you can Mark done the completed ones and also Delete note When viewing a note in detail, it is possible to start Editing it (change the name, the description, and the tags). The sidebar can search for notes by tags and by the Done/Undone field.
    {# Web-scrapper block #}
  • Web-scrapper

    • After authentication, the user has an opportunity to view current news in the following categories:

    • Ukrainian news
    • Sports news
    • Technology news
    • Current books on the Python programming language
    • Current exchange rates in banks

    The news, the book titles, and the bank names have links to the corresponding pages that you can go to for more detailed information.

    To display the exchange rate, select the required currency and date to get the corresponding report.

  • Profile
    • After successful signing up or signin in the user will be redirected straight to their profile where they can add their first name and last name, set avatar, email, phone, and gender. Also, the user is able to see how many contacts, notes, and files they have.
