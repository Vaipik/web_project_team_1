This website has been created as team final project in GoIT-PyWeb7 course. It consists from 6 applications.
    {# Auth block #}
  • Authentication
    • To use this website user must be authenticated. The only page that can be accessed by unauthorized user is this page and sign up page.

    On sign up page user are able to sign up and sign in using modal login form on the top right corner. Although on that corner user are able to sign out.

    {# Contacts block #}
  • Contacts
    • User able to do smth
    {# File storage block #}
  • File storage
    • User able to upload files with valid extensions. Valid xtensions can be found in header on the files page. Just click on button Extensions. Uploaded files are separated into categories according to their extensions and stored in the DropBox cloud. User are able to filter files by category.
    {# Notes block #}
  • Notes
    • User able to add notes just click on button Add note. Each note must include a tag that describes the topic and subject of the entry. In the Tags section, you can create new tags and delete those that are not used. In the list of notes, you can Mark done that are completed and also Delete note When viewing a note in detail, it is possible to start Editing it (change the name, description, and tags). The sidebar has the ability to search for notes by tags and by the Done/Undone field
    {# Web-scrapper block #}
  • Web-scrapper

    • After authentication, the user has the opportunity to view current news in the following categories:

    • Ukrainian news
    • Sports news
    • Technology news
    • Current books on the Python programming language
    • Current exchange rates in banks

    News, book titles, and bank names have links to the corresponding pages that you can go to for more detailed information.

    To display the exchange rate, select the required currency and date for the corresponding report.

    {# Search block #}
  • Search
    • User able to do smth
    {# Profile block #}
  • Profile
    • After successful signing up or signin in user will be redirect directly to his profile where user profile can be edited by adding obligatory fields: first name and last name or not obligatory such as avatar, email, phone, gender. Although user are able to see how much contacts, notes and files it has.
